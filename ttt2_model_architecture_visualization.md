## 詳細模組說明

### 1. 預訓練 WavTokenizer 編碼器
```ascii
輸入音頻 [B, 1, T]
        │
        ▼
┌─────────────────┐
│   Encodec       │
│   Encoder       │  → 特徵提取
│   (凍結)        │
└─────────────────┘
        │
        ▼
特徵 [B, 512, T] + 離散編碼
```

**功能**: 將原始音頻波形轉換為高維特徵表示
**參數**: 80,683,492 (凍結，不可訓練)
**輸出**: 512維特徵 + 離散量化編碼

### 2. 特徵增強變換鏈

#### 2.1 輸入處理層
```ascii
Features [B, 512, T]
        │
        ▼
┌─────────────────┐
│   LayerNorm     │  歸一化輸入特徵
│    (512)        │
└─────────────────┘
        │
        ▼
┌─────────────────┐
│   Conv1d        │  降維投影
│  512→256, k=1   │  減少計算量
│  + BatchNorm    │
│  + GELU         │
└─────────────────┘
        │
        ▼
┌─────────────────┐
│ Dropout(0.25)   │  防止過擬合
└─────────────────┘
```

#### 2.2 殘差塊架構 (核心變換)
```ascii
                   殘差塊 × 5 層
                        
輸入 [B, 256, T]            每一層都包含:
        │                    
        ├──────────────────┐  • Conv1d(256→256, k=3, p=1)
        │                  │  • GroupNorm(8 groups, 256 channels)  
        ▼                  │  • GELU 激活
   ┌──────────┐            │  • Dropout(0.35)
   │ Conv1d   │            │  • Conv1d(256→256, k=3, p=1)
   │k=3, p=1  │            │  • GroupNorm(8 groups, 256 channels)
   └──────────┘            │  • 殘差連接
        │                  │  • GELU 激活
        ▼                  │
   ┌──────────┐            │  中間特徵會在每層後保存
   │GroupNorm │            │  用於內容一致性損失計算
   │ (8,256)  │            │
   └──────────┘            │
        │                  │
        ▼                  │
   ┌──────────┐            │
   │   GELU   │            │
   └──────────┘            │
        │                  │
        ▼                  │
   ┌──────────┐            │
   │ Dropout  │            │
   │  (0.35)  │            │
   └──────────┘            │
        │                  │
        ▼                  │
   ┌──────────┐            │
   │ Conv1d   │            │
   │k=3, p=1  │            │
   └──────────┘            │
        │                  │
        ▼                  │
   ┌──────────┐            │
   │GroupNorm │            │
   │ (8,256)  │            │
   └──────────┘            │
        │                  │
        ▼                  │
   ┌──────────┐            │
   │    +     │←───────────┘
   │ (殘差加法) │
   └──────────┘
        │
        ▼
   ┌──────────┐
   │   GELU   │
   └──────────┘
        │
        ▼
   輸出 [B, 256, T]
```

**關鍵特性**:
- **GroupNorm**: 使用8個群組，提升穩定性，相比BatchNorm對批次大小不敏感
- **GELU激活**: 平滑的非線性激活，比ReLU更適合特徵表示學習
- **殘差連接**: 防止梯度消失，允許深度網絡訓練
- **Dropout**: 0.35的比率提供強正則化

#### 2.3 輸出處理層
```ascii
Enhanced [B, 256, T]
        │
        ▼
┌─────────────────┐
│   Conv1d        │  升維恢復
│  256→512, k=1   │  匹配解碼器輸入
│  + BatchNorm    │
│  + GELU         │
└─────────────────┘
        │
        ▼
┌─────────────────┐
│   LayerNorm     │  最終歸一化
│    (512)        │
└─────────────────┘
        │
        ▼
Enhanced Features [B, 512, T]
```

### 3. 預訓練 WavTokenizer 解碼器
```ascii
Enhanced Features [B, 512, T]
        │
        ▼
┌─────────────────┐
│   Encodec       │
│   Decoder       │  → 音頻重建
│   (凍結)        │
└─────────────────┘
        │
        ▼
Denoised Audio [B, 1, T]
```

**功能**: 將增強特徵重建為去噪音頻
**參數**: 凍結，保持預訓練權重

## 訓練流程

### 損失函數架構
```ascii
                    損失計算流程
                        
Enhanced Features      Target Features
      │                      │
      │                      │
      ▼                      ▼
┌─────────────────────────────────────┐
│        特徵空間 L2 損失              │
│   L2_loss = ||E - T||₂             │
│                                     │
│   權重: β = 0.99 (主要損失)          │
└─────────────────────────────────────┘
      │
      ▼
┌─────────────────────────────────────┐
│       內容一致性損失                │
│                                     │
│  使用中間特徵 (第2層殘差塊輸出)     │
│  確保相同內容ID的樣本特徵相似        │
│                                     │
│   權重: α = 0.01 (輔助損失)         │
└─────────────────────────────────────┘
      │
      ▼
┌─────────────────────────────────────┐
│           總損失                    │
│   Total = α × Content + β × L2     │
└─────────────────────────────────────┘
```

### 訓練參數配置
```ascii
優化器配置:
├── AdamW優化器
│   ├── 學習率: 1e-4
│   ├── 權重衰減: 1e-5
│   └── β: (0.9, 0.999)
│
├── 學習率調度器
│   ├── ReduceLROnPlateau
│   ├── 監控: 驗證損失
│   ├── 降低因子: 0.5
│   └── 耐心值: 5 epochs
│
└── 正則化
    ├── Dropout: 0.25 (輸入) + 0.35 (殘差塊)
    ├── GroupNorm: 提升穩定性
    └── 權重衰減: 防止過擬合
```

```ascii
內容一致性損失計算:
│

                    ▼
          提取第2層殘差塊特徵
                    │
                    ▼
     計算相同content_id樣本的特徵相似性
                    │
                    ▼
            鼓勵特徵聚類對齊
```

```ascii
特徵保存策略:
│

│
用途:
├── t-SNE視覺化
```
