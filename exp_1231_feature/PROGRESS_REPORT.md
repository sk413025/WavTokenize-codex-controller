## 1. 研究背景與動機

### 1.1 問題描述

在使用 LoRA 微調 WavTokenizer encoder 進行音頻降噪任務時，觀察到：

| 現象 | 數值 |
|------|------|
| Train Accuracy | 持續上升 (3.22%) |
| Val Accuracy | 停滯不前 (1.06%) |
| **Train-Val Gap** | **2.33%** (過擬合徵兆) |

### 1.2 核心問題

> **模型是否「學歪了」？**
> 
> 類比圖像分類任務：
> - 正常模型: Feature map 在「貓的輪廓」發光 → 正確分類
> - 學歪模型: Feature map 在「背景沙發」發光 → 過擬合，無法泛化

### 1.3 研究方法

透過 Feature Map 分析，回答四個關鍵問題：

| # | 問題 | 分析方法 |
|---|------|----------|
| 1 | WavTokenizer 各層在看什麼？ | Speaker Invariance 分析 |
| 2 | LoRA 訓練後哪些層變化最大？ | Weight Change 分析 |
| 3 | Train/Val 是否關注相同特徵？ | Attention Pattern 分析 |
| 4 | 各層對噪音的敏感度如何？ | Noise Sensitivity 分析 |

## 3. Feature Map 分析實驗

### 3.1 分析一：Speaker Invariance（說話者不變性）

**實驗方法**: 比較不同說話者 (boy1 vs girl2) 說同一句話時，各層 feature map 的 cosine similarity

**實驗假設**: 
- 淺層提取聲學特徵 → 不同說話者差異大，similarity 低
- 深層提取語義特徵 → 同一內容趨於一致，similarity 高

#### 結果

| 層組 | 平均 Cosine Similarity | 解讀 |
|------|------------------------|------|
| Input (L0) | -0.011 | 完全不同 |
| Low-Level (L1-L4) | 0.017 | 幾乎無關 |
| Mid-Level (L5-L8) | 0.073 | 略有相關 |
| Semantic (L9-L12) | 0.118 | 開始收斂 |
| **Abstract (L13-L16)** | **0.427** | **高度相似** |
| Output (L17) | 0.187 | 投影後略降 |

**關鍵數據**:
- L14 達到最高 similarity: **0.67**
- L13 次高: **0.52**

> **📊 圖表位置**: `exp_1231_feature/outputs/speaker_invariance_summary.png`

#### 視覺化示意

```

```

**結論**: ✅ WavTokenizer 的層級結構符合預期，淺層處理聲學，深層處理語義

### 3.3 分析三：Train/Val Attention Consistency（注意力一致性）

**實驗方法**: 
1. 從 train/val 各取 20 個樣本
2. 計算各層的 channel activation pattern
3. 比較 Train/Val 的 pattern similarity

**核心問題**:
> Train 和 Val 是否關注「相同類型」的特徵？  
> 還是一個在看「貓輪廓」，一個在看「背景沙發」？

#### 結果

| 指標 | 數值 | 解讀 |
|------|------|------|
| **Pattern Similarity** | **0.9907** | ✅ 極高！關注相同特徵 |
| CKA | 0.3209 | 中等（不同說話者，絕對值不同是正常的） |

> **📊 圖表位置**: `exp_1231_feature/outputs/attention_consistency.png`

**結論**: ✅ **排除資料問題** — Train 和 Val 確實關注相同類型的特徵

## 4. 驗證實驗：凍結深層與分階段訓練

### 4.1 實驗設計

基於 Feature Map 分析結論，設計兩個驗證實驗：

| 實驗 | 策略 | 假設 |
|------|------|------|
| **Exp B** | 凍結深層 (L9-L17) | 如果深層變化是問題，凍結應消除 Gap |
| **Exp E** | 分階段訓練 | 先讓淺層學會去噪，再逐步解凍深層 |

### 4.2 Exp B: 凍結深層

**配置**:
- LoRA 層: L0-L8 (9 層)
- 凍結層: L9-L17 (9 層)
- Rank: 256
- 訓練: 300 epochs

### 4.3 Exp E: 分階段訓練

**配置**:
```
Phase 1 (Epoch 1-100):   只訓練 L0-L4 (5 層)
Phase 2 (Epoch 101-200): 訓練 L0-L8 (9 層)
Phase 3 (Epoch 201-300): 訓練 L0-L17 (18 層)
```

### 4.4 實驗結果比較

| 指標 | Baseline | Exp B (凍結深層) | Exp E (分階段) |
|------|----------|------------------|----------------|
| **Best Val Acc** | **1.06%** | 0.51% | 0.82% |
| **Train-Val Gap** | 2.33% | **0.06%** | 1.49% |
| 結論 | 過擬合 | Gap 消除但 Acc 崩潰 | Gap 改善 |

> **📊 圖表位置**: 
> - `exp_1231_1/runs/exp_b_freeze_deep/training_curves.png`
> - `exp_1231_1/runs/exp_e_progressive/training_curves.png`

```

│                         重要發現                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  Exp B 結果分析:                                                     │

## 5. 差異化訓練策略

### 5.1 策略設計

基於分析結論，設計三種差異化訓練策略：

| 實驗 | 策略 | 原理 |
|------|------|------|
| **Exp F** | 差異化學習率 | 淺層 LR=1e-4 (10x)，深層 LR=1e-5 (1x) |
| **Exp G** | 差異化 Rank | 淺層 rank=256，深層 rank=32 |
| **Exp H** | L2 正則化 | 對深層權重變化施加懲罰 |

### 5.2 Exp F: 差異化學習率

**配置**:
```
淺層 (L0-L8):  LR = 1e-4  (基準的 10 倍)
深層 (L9-L17): LR = 1e-5  (基準)
```

**預期**: 強制淺層多學習，限制深層變化

### 5.3 Exp G: 差異化 Rank

**配置**:
```
淺層 (L0-L8):  Rank = 256, Alpha = 512

## 6. 結論與未來方向

### 6.1 主要發現總結

| # | 發現 | 意義 |
|---|------|------|
| 1 | WavTokenizer 層級結構健康 | 淺層處理聲學，深層處理語義 |
| 2 | 噪音主要影響 Mid-level (L5-L6) | 確定去噪應聚焦的層 |
| 3 | LoRA 深層變化大是「症狀」 | 真因是淺層學習不足 |
| 4 | Train/Val 關注相同特徵 | 排除資料問題 |
| 5 | 需要協同訓練而非凍結 | 凍結破壞 Accuracy |
| 6 | 差異化策略可改善 Gap | 有效方向 |

### 6.2 Train-Val Gap 問題的完整解釋

```
┌─────────────────────────────────────────────────────────────────────┐
│                    Train-Val Gap 問題的完整機制                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  1. 標準 LoRA 訓練時:                                                │
│     • 淺層變化不足 → 沒有有效去噪                                    │
│     • 深層被迫補償 → 「硬記」訓練集的 noisy→clean 映射              │
│                                                                      │
│  2. 這種補償的特性:                                                  │
│     • 在訓練集上有效（記住了特定噪音模式）                          │
│     • 在驗證集上失敗（噪音模式/說話者不同）                         │
│     • 表現為 Train Acc 上升，Val Acc 停滯                           │
│                                                                      │
│  3. 正確的解決方向:                                                  │
│     • 強制淺層多學習（差異化 LR/Rank）                              │
│     • 限制深層變化（低 Rank/正則化）                                │
│     • 保持協同訓練（不完全凍結）                                    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### 6.3 未來方向

   - 觀察是否超越 Baseline 的 1.06%
   - 確認 Gap 控制效果

2. **組合策略**
   - 如果 Exp F/G 成功，嘗試結合兩者
   - 差異化 LR + 差異化 Rank

3. **進階分析**
   - 分析 Teacher vs Student 特徵差異
   - 追蹤各層權重在訓練過程中的變化軌跡
