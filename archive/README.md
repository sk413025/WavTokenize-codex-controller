# Archive Surface

This directory holds material that should stay available for traceability but should no longer occupy the repo's main navigation path.

## What Lives Here
- `root-notes/`: one-off reports, thesis notes, weekly summaries, and upstream explanation notes
- `root-scripts/`: stray root-level helper or backup files
- `legacy_artifacts/`: legacy result trees, backups, and archived experiment bundles
- `legacy_families/`: early or retired `exp_*` experiment families that are no longer part of the main repo navigation path
- `controller_runs/`: dry-runs, demos, and non-canonical run ledgers moved out of the main official evidence path

Archive content is not part of the official start path.

## Archive Boundaries
- `controller_runs/` here are kept for traceability only.
- Do not treat `archive/controller_runs/*` as the current official evidence path for live status, promotion, or follow-up decisions.
- The canonical current run ledger remains `controller_runs/<run_id>/` at the repo root when a run is still part of the active official surface.
