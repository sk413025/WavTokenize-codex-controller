# Exp 0106: 差異化訓練策略實驗

## 背景

基於 `exp_1231_1` 實驗結論設計的後續實驗。

### 前置實驗結果

| 實驗 | 策略 | Best Val Acc | Train-Val Gap | 結論 |
|------|------|-------------|---------------|------|
| Baseline (exp64) | 全層 LoRA | 1.06% | 2.33% | 深層變化大，過擬合 |
| Exp B | 凍結深層 L9-L17 | 0.51% | 0.06% | Gap 消除，但準確度崩潰 |
| Exp E | 漸進訓練 | 0.82% | 1.49% | Gap 改善，但進步有限 |

### 關鍵洞察

從 Exp B 和 Exp E 的結果得出重要結論：

1. **深層變化大是「症狀」不是「原因」**
   - 凍結深層確實消除了 Train-Val Gap
   - 但同時也破壞了準確度

2. **真正問題：淺層學習不足**
   - 淺層 (L0-L8) 學習不夠 → 無法有效去噪
   - 深層 (L9-L17) 被迫補償 → 過度變化 → 過擬合

3. **正確的信號流**
   ```
   noisy audio
        │
        ▼
   ┌──────────────────────────────────────────┐
   │ 淺層 (L0-L8) - 應該多學習，識別並修正噪音 │
   └─────────────────────┬────────────────────┘
                         │
   ┌──────────────────────────────────────────┐
   │ 深層 (L9-L17) - 只需微調以接收修正後特徵   │
   └─────────────────────┬────────────────────┘
                         ▼
                    VQ Codebook
   ```

## 實驗設計

### Exp F: 差異化學習率 (`--model_type diff_lr`)

**策略**：強制淺層多學習，限制深層變化

| 層組 | 學習率 | 比例 |
|------|--------|------|
| 淺層 (L0-L8) | 1e-4 | 10x |
| 深層 (L9-L17) | 1e-5 | 1x |

**預期**：
- 淺層獲得更多學習機會，能更有效去噪
- 深層變化受限，減少過擬合
- Train-Val Gap 應該縮小，同時保持準確度

**執行**：
```bash
bash exp_0106/run_exp_f.sh
```

---

### Exp G: 差異化 Rank (`--model_type diff_rank`)

**策略**：淺層高容量，深層低容量

| 層組 | LoRA Rank | Alpha | 比例 |
|------|-----------|-------|------|
| 淺層 (L0-L8) | 256 | 512 | 8x |
| 深層 (L9-L17) | 32 | 64 | 1x |

**預期**：
- 淺層有足夠容量學習複雜的去噪模式
- 深層容量受限，只能做簡單微調
- 從結構上限制深層的變化能力

**執行**：
```bash
bash exp_0106/run_exp_g.sh
```

---

### Exp H: L2 正則化 (`--model_type l2_reg`)

**策略**：全層訓練，但對深層施加額外懲罰

| 層組 | LoRA Rank | L2 權重 |
|------|-----------|---------|
| 淺層 (L0-L8) | 256 | 0 |
| 深層 (L9-L17) | 256 | 0.1 |

**機制**：
- L2 損失 = `||W_current - W_initial||²`
- 懲罰深層 LoRA 權重偏離初始值太遠
- 動態平衡：允許必要的調整，限制過度變化

**預期**：
- 更靈活的正則化方式
- 深層可以學習，但變化幅度受限
- 可能比 Exp F/G 更能保持準確度

**執行**：
```bash
bash exp_0106/run_exp_h.sh
```

## 實驗對比

| 實驗 | 淺層策略 | 深層策略 | 優點 | 缺點 |
|------|---------|---------|------|------|
| Exp F | 高 LR (1e-4) | 低 LR (1e-5) | 簡單直接 | 深層學習可能太慢 |
| Exp G | 高 Rank (256) | 低 Rank (32) | 結構性限制 | 可能限制太死 |
| Exp H | 正常 | L2 正則化 | 靈活調節 | 需要調參 λ |

## 評估指標

1. **Val Masked Accuracy** - 主要指標，目標 > 1.06% (baseline)
2. **Train-Val Gap** - 目標 < 2.33% (baseline)，理想 < 1%
3. **收斂速度** - 達到最佳性能所需 epoch
4. **深層權重變化** - 監控深層 LoRA 權重變化幅度
