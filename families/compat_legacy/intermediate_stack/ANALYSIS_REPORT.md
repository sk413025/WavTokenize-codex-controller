## 三、詳細分析

### 3.1 為何 v3 沒有超越 v2？

#### 假說 1：L5 + L10 增加了學習負擔

v3 增加了兩個監督點：
- **L5 (w=0.8)**: 新增的中間層監督
- **L10 (w=0.3)**: MSE 語義錨點

但 Total Loss 顯著增加：
- v2 Final Val Loss: 1.941
- v3 Final Val Loss: 2.751 (+42%)

**原因分析**：
```
v3 的 Intermediate Loss 組成更複雜：
L3:  0.5 × 0.770 = 0.385
L5:  0.8 × 0.488 = 0.390  ← 新增
L6:  1.0 × 0.802 = 0.802  ← 權重提高
L10: 0.3 × 0.0008 = 0.0002  ← 新增 (但很小)
─────────────────────────────
Total weighted: ~1.58 (vs v2 的 ~0.77)
```

v3 的中間層監督總權重約為 v2 的 2 倍，這可能分散了模型對主任務的學習。

#### 假說 2：L6 權重過高

v2: L6 weight = 0.5
v3: L6 weight = 1.0 (加倍)

結果 L6 的 Val Loss 反而更高：
- v2 L6 Val Loss: 0.774
- v3 L6 Val Loss: 0.802 (+3.6%)

**可能原因**：權重過高導致過度擬合該層，但未能轉化為更好的下游性能。

#### 假說 3：L10 MSE 錨點效果有限

L10 的 MSE Loss 非常小 (0.0008)，說明：
1. L10 本來就很穩定，不需要強監督
2. MSE Loss 的貢獻幾乎可忽略

### 3.2 正面發現

儘管 v3 整體準確率略低，但有一個正面指標：

**Train-Val Gap 減少了 10%**
- v2: 2.589%
- v3: 2.332%

這表示 v3 的**泛化能力可能更好**，只是需要更多訓練或調整權重配置。

### 3.3 各層中間監督效果

| 層 | v2 Val Loss | v3 Val Loss | 變化 | 解讀 |
|----|-------------|-------------|------|------|
| L3 | 0.768 | 0.770 | +0.3% | 持平 |
| L5 | N/A | 0.488 | - | 新增，收斂良好 |
| L6 | 0.774 | 0.802 | +3.6% | 略差 |
| L10 | N/A | 0.0008 | - | 極小，效果存疑 |

**L5 的收斂非常好** (0.488 vs L3 的 0.77, L6 的 0.80)，這符合預期——L5 是噪音處理的協同層。

## 五、結論與建議

### 5.1 核心結論

| 結論 | 說明 |
|------|------|
| **v2 ≈ v3** | 準確率差異 < 0.01%，統計上可能不顯著 |
| **4 層監督未帶來提升** | 增加監督點不一定更好 |
| **L5 收斂良好** | 但未轉化為更好的下游性能 |
| **L10 錨點效果有限** | MSE Loss 太小，幾乎無貢獻 |
| **v3 泛化略好** | Train-Val Gap 減少 10% |

### 5.2 可能的改進方向

#### 方案 A：降低中間層總權重
```python
# 當前 v3
intermediate_weight = 1.0  # 總開關
L3=0.5, L5=0.8, L6=1.0, L10=0.3

# 建議：降低總權重
intermediate_weight = 0.5  # 或 0.3
# 保持內部比例不變
```

#### 方案 B：簡化為 3 層監督
```python
# 移除 L10（效果有限）
intermediate_indices = [3, 5, 6]
intermediate_weights = {3: 0.5, 5: 0.8, 6: 0.8}  # L6 降回 0.8
```

#### 方案 C：調整 L6 權重
```python
# L6 權重過高，嘗試降回
L6_weight = 0.5  # 與 v2 相同
# 或嘗試
L6_weight = 0.7  # 折衷
```

#### 方案 D：增加訓練長度
v3 在 Epoch 260 才達到最佳，可能需要更多訓練時間。

### 5.3 建議的下一步實驗

| 優先級 | 實驗 | 配置 | 目標 |
|--------|------|------|------|
| **高** | Exp K v4 | L3(0.5) + L5(0.8) + L6(0.5) | 測試 L6 權重影響 |
| 中 | Exp K v5 | v3 配置 + intermediate_weight=0.5 | 降低總權重 |
| 低 | Exp K v6 | 只監督 L5, L6 | 測試最小配置 |
