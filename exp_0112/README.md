# Exp 0112: 三區差異化訓練 (Exp I)

## 實驗目標

基於之前實驗的關鍵發現，設計「中層強化」訓練策略：

### 實驗背景

| 實驗 | 策略 | 結果 | 教訓 |
|------|------|------|------|
| Exp B | 凍結深層 | Val Acc 崩潰 (-52%) | 深層必須保持調整能力 |
| Exp F | 淺層高 LR (10x) | Gap 改善但 Acc 下降 | 二分法太粗糙 |
| Exp G/H | 深層 L2 正則 | Gap 反而惡化 | 限制深層方向錯誤 |

### 關鍵洞察

1. **中層 (L5-L8) 對噪音最敏感** (cos similarity = 0.21-0.29)
2. **深層調整是必要條件**，不是過擬合原因
3. 需要「協同訓練」但「差異化強度」

## Exp I: 三區差異化配置

### 核心設計

```
┌─────────────────────────────────────────────────────────────────────┐
│                      三區差異化訓練策略                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  淺層 (L0-L4): 聲學特徵層                                           │
│  ├─ LR = 5e-6 (0.5x base)                                          │
│  ├─ L2 = 0.01 (輕度約束)                                           │
│  └─ 目標: 穩定基礎聲學表示，避免過度變化                           │
│                                                                      │
│  中層 (L5-L8): 噪音敏感層 ★ 重點區域                               │
│  ├─ LR = 2e-5 (2.0x base)                                          │
│  ├─ L2 = 0 (無約束)                                                │
│  └─ 目標: 最大化學習去噪能力                                       │
│                                                                      │
│  深層 (L9-L17): 語義特徵層                                          │
│  ├─ LR = 1e-5 (1.0x base)                                          │
│  ├─ L2 = 0 (無約束)                                                │
│  └─ 目標: 自由調整以配合上游變化                                   │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### 與之前實驗對比

| 配置項 | Baseline | Exp F | Exp I (新) |
|--------|----------|-------|------------|
| 淺層 LR | 1e-5 | 1e-4 (10x) | **5e-6** (0.5x) |
| 中層 LR | 1e-5 | 1e-4 (10x) | **2e-5** (2.0x) |
| 深層 LR | 1e-5 | 1e-5 (1x) | 1e-5 (1.0x) |
| 淺層 L2 | 0 | 0 | **0.01** |
| 中/深層 L2 | 0 | 0 | 0 |
| 分區 | 無 | 二分 | **三分** |

### 預期效果

1. **中層強化學習**: 高 LR 讓噪音敏感區有足夠學習動力
2. **淺層穩定**: 低 LR + L2 防止基礎特徵被破壞
3. **深層自由配合**: 保持梯度流通，允許適度調整
4. **避免 Exp F 問題**: 不會因淺層 LR 過高導致不穩定
