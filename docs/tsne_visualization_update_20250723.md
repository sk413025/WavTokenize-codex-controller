# t-SNE 可視化更新 (2025-07-23)

## 功能更新說明

我們對 t-SNE 可視化功能進行了重要更新，使其能夠以更有意義的方式呈現訓練資料的特徵表示。

### 更新內容

1. **每句話為一個獨立特徵點**：
   - 先前的實現方式可能混合了不同句子和時間幀的特徵
   - 新的實現將每句話的特徵在時間維度上平均，以生成一個代表整句話的特徵向量
   - 每個點現在代表一句完整的語音，而不是單個時間幀

2. **增強對比可視化**：
   - 使用不同形狀標記區分增強特徵（圓形）和原始特徵（方形）
   - 對於少量資料點的情況（≤30個點），添加索引標籤以便追蹤
   - 改進圖表標題和標籤，明確指出每個點代表一句話

3. **特徵處理方式**：
   - 特徵在時間維度上取平均，保留批次和通道維度
   - 保持完整的句子級別特徵，不再分解為時間幀
   - 標準化處理以便更好地比較原始和增強特徵

## 可視化解讀指南

新的 t-SNE 可視化圖應按照以下方式解讀：

- **藍色圓點**：經過模型處理後的增強特徵，每個點代表一句完整的語音
- **紅色方塊**：原始的目標特徵，每個點代表一句完整的語音
- **點的聚集度**：表示特徵空間中語音的相似性
- **點對的距離**：相同索引的藍點和紅點間距離越小，表示模型對該句話的特徵還原越好

## 科學價值

這種可視化方式能夠更準確地評估模型的學習能力：

1. **整體分布評估**：觀察增強特徵和目標特徵的整體分布是否相似
2. **詳細句子比較**：比較相同句子在處理前後的變化程度
3. **異常識別**：容易發現哪些句子的處理效果特別好或特別差
4. **學習趨勢分析**：通過比較不同訓練階段的可視化圖，追蹤模型的學習進程

## 技術實現

更新是通過修改 `plot_tsne_visualization()` 函數實現的，主要變更包括：

1. 特徵在時間維度上平均化
2. 使用不同的標記形狀來區分增強和原始特徵
3. 為少量樣本添加索引標籤
4. 更新標題和標籤說明

此更新將在下一次訓練迭代中生效，所有新生成的 t-SNE 圖將採用這種呈現方式。
