# t-SNE 可視化更新：每句話為一個特徵點

## 更新日期：2025-07-23

## 概述

為了更好地評估和展示模型的學習能力，我們對 t-SNE 可視化功能進行了重要更新。現在，每個特徵點代表一句完整的語音，而不是單個時間幀，這使我們能夠更清晰地觀察模型對每句話的處理效果。

## 主要改進

1. **句子級別特徵表示**
   - 將每句話在時間維度上取平均，生成一個統一的特徵向量
   - 每個點現在代表一句完整的語音，使可視化更有意義

2. **增強的視覺效果**
   - 使用連線連接相同句子的原始和增強特徵點
   - 連線顏色根據t-SNE空間中的距離變化，提供直觀的相似度指標
   - 添加顏色條顯示距離比例
   - 為小樣本量添加句子索引標籤

3. **統計信息整合**
   - 計算和顯示點對之間的平均距離和標準差
   - 提供整體模型效果的量化指標

## 技術實現

更新主要涉及 `plot_tsne_visualization()` 函數的重寫，主要變更包括：

1. 在時間維度上取平均，將 [N, C, T] 形狀的特徵轉換為 [N, C]
2. 添加句子對之間的連線和距離顏色映射
3. 計算和顯示點對之間的統計距離信息
4. 改進標題、標籤和圖例

## 使用方法

現有的訓練流程無需更改即可使用此功能。每150個epoch生成的t-SNE可視化現在將使用這種句子級別的表示方式。

此外，我們提供了一個專門的測試腳本：

```bash
./test_sentence_tsne.sh
```

此腳本將生成三種測試圖像：
1. 小樣本測試 (10句話)
2. 中等樣本測試 (50句話)
3. 距離漸變測試 (模擬不同程度的特徵變化)

## 解讀指南

新的 t-SNE 可視化應按以下方式解讀：

- **藍色圓點**：模型處理後的增強特徵
- **紅色方塊**：原始目標特徵
- **連線顏色**：較淺顏色表示點對距離較小，模型保留了更多原始特徵
- **點的分布**：相似句子會在特徵空間中聚集
- **平均距離**：整體模型效果的量化指標，值越小表示模型還原效果越好

## 預期收益

1. **更精確的模型評估**：通過觀察每句話的特徵變化，更準確地評估模型效果
2. **問題診斷**：容易識別模型處理效果特別好或特別差的句子
3. **直觀的進展可視化**：通過比較不同訓練階段的可視化，直觀追蹤模型的學習進展
4. **更有意義的特徵空間理解**：了解模型如何在句子級別組織特徵空間

## 後續規劃

1. 添加更多視覺化元素，如句子內容標籤或音頻特性指示
2. 實現交互式可視化，支持點擊聽取相應音頻
3. 增加不同模型和訓練階段的比較功能
