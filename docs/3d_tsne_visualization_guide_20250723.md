# 3D t-SNE 特徵可視化功能使用指南

**日期：2025-07-23**

## 功能概述

3D t-SNE 可視化功能是對現有分層特徵可視化系統的強化，通過將特徵投射到三維空間，提供更全面的特徵結構分析視角。相比二維可視化，三維可視化能更好地保留原始高維特徵空間的結構信息，減少降維過程中的信息損失。

## 主要功能

1. **單層 3D t-SNE 可視化**
   - 為每個殘差塊層生成單獨的 3D t-SNE 可視化
   - 從多個角度（0°, 45°, 90°, 135°）保存視圖
   - 顯示特徵點與目標特徵的空間關係

2. **全層 3D 比較**
   - 在同一個 3D 空間中比較不同層的特徵分布
   - 使用漸變色彩區分不同層級
   - 便於觀察特徵從早期層到後期層的演變過程

3. **360° 旋轉動畫**
   - 生成特徵空間的旋轉視圖
   - 以 GIF 格式保存，便於在報告中引用
   - 提供全方位的特徵空間觀察角度

## 使用方法

### 啟用 3D 可視化

在 `run_layer_visualization.sh` 腳本中，將環境變數設定如下：

```bash
export ENABLE_3D_TSNE=true  # 啟用3D t-SNE可視化
```

若不需要 3D 可視化，可設定為 `false` 或移除此環境變數。

### 執行分析

```bash
bash run_layer_visualization.sh
```

### 可視化結果

運行完成後，可在以下目錄找到 3D 可視化結果：

```
results/layer_viz_{實驗ID}/layers/3d/
```

結果包含：
- 每層的多角度 3D t-SNE 圖（`tsne_3d_layer_X_angle_Y.png`）
- 全層比較的多角度視圖（`tsne_3d_all_layers_angle_Y.png`）
- 特徵空間旋轉動畫（`tsne_3d_rotation.gif`）

## 技術細節

- 特徵標準化：使用 `StandardScaler` 進行特徵標準化
- 降維方法：使用 `sklearn.manifold.TSNE` 的 3D 模式
- 優化參數：
  - `perplexity`：動態調整，基於樣本數量
  - `init`：使用 PCA 初始化以加速收斂
  - `learning_rate`：自動選擇
  - `n_iter`：使用 2000 次迭代以確保收斂

## 結果解讀

### 單層 3D 可視化
- 藍色點：對應層的特徵
- 紅色點：目標特徵
- 灰色線：連接對應的特徵點，表示特徵變換關係
- 距離指標：提供類內分散度、中心距離和平均位移的量化評估

### 全層 3D 比較
- 藍到紅的漸變：從早期層到晚期層的顏色映射
- 紅色點：目標特徵
- 整體分布：觀察特徵空間如何隨著層深演變

## 常見問題

1. **記憶體使用過高**
   - 減少批次大小：設置 `TTT_BATCH_SIZE=2`
   - 降低樣本數量：修改 `max_points` 參數

2. **3D 旋轉動畫無法創建**
   - 確保安裝了 `imageio` 套件：`pip install imageio`
   - 檢查系統是否有足夠的臨時空間

3. **可視化質量不佳**
   - 增加 t-SNE 迭代次數：修改 `n_iter` 參數
   - 嘗試不同的 `perplexity` 值
